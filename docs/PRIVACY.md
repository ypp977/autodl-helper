# Privacy and local data handling

`autodl-helper` is designed to run locally. It does **not** include a hosted backend or a built-in remote telemetry pipeline.

## What is stored locally

By default, the project stores local runtime data in these locations:

- `config.yaml`
  - local project configuration
- `data/autodl-helper.db`
  - SQLite history, runtime control state, keeper history, scheduled history, event logs
- `.cache/*.json`
  - auth cache files
- `logs/`
  - daemon stdout/stderr and related runtime logs
- `.autodl-helper-auth.json`
  - local auth-related state if generated by local runtime flow
- `.autodl-helper-state.json`
  - local runtime state file
- `.autodl-helper.lock`
  - local lock file

## Sensitive information you may store locally

Depending on how you configure the tool, local files may contain:

- AutoDL authorization tokens
- AutoDL phone numbers
- instance IDs
- machine preferences
- runtime history and timestamps

Treat the following as sensitive:

- `.env`
- `config.yaml`
- `data/`
- `logs/`
- `.cache/`
- `.autodl-helper-*.json`

## What should never be committed

Do **not** commit:

- `.env`
- `config.yaml`
- `data/`
- `logs/`
- `.cache/`
- `.autodl-helper-auth.json`
- `.autodl-helper-state.json`
- `.autodl-helper.lock`

Use:

- `config.example.yaml`
- `.env.template`

for public-safe examples only.

## Screenshots and issue reports

Before publishing screenshots, issue reports, or logs:

- remove tokens
- remove phone numbers
- remove real instance IDs if you consider them private
- review terminal output for account identifiers

## Network behavior

This project talks to AutoDL-related endpoints as part of its intended functionality. It does not currently document or implement a separate analytics service.

If you need stricter guarantees for your environment, review the source directly before use.

## Recommendation before open-sourcing your fork

Before pushing public history:

1. inspect screenshots in `images/`
2. inspect git history for leaked tokens or phone numbers
3. rotate credentials if you suspect they were ever committed
